You can run the game in two ways:

Option 1: Downloading a Prebuilt File

The game is available for download in versions for Windows and Linux:

Windows
  1. Download the .exe file.
  2. Double-click the downloaded .exe file to launch the game.
  3. The game will start immediately.

Note: If you see a warning from your antivirus or Windows Defender, this is a standard alert for new or unsigned files. The game is safe, and the warning can be ignored.

Linux
  1. Download the .x86_64 file.
  2. Open a terminal and navigate to the directory containing the file.
  3. Grant execution permissions by typing: 
    chmod +x PanDa5.x86_64
  4. Run the game by typing: 
    ./PanDa5.x86_64

Note: No additional installation or libraries are required to run the game.

Option 2: Cloning the Repository
  1. Make sure you have Godot Engine installed. You can download it from Godot's official website.
  2. Clone the repository using the following command: 
    git clone https://github.com/akotu235/panda5.git
  3. Navigate to the project directory: 
    cd panda5
  4. Open the project in Godot by selecting the project.godot file.
  5. Run the game by clicking the "Run" button in the Godot editor.

This option is ideal for those who want to explore or modify the game's source code.
